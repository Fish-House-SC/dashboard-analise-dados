# dashboard-analise-dados
Projeto desenvolvido para análise de dados com visualização interativa.

## Funcionalidades

- Upload de arquivos Excel ou CSV
- Filtro dinâmico de dados
- Gráficos interativos (Plotly)
  - Linha
  - Barra
  - Histograma
- Métricas automáticas (média, máximo, mínimo)
- Download dos dados filtrados (Excel/CSV)
- Temas visuais personalizáveis

## Tecnologias

- Python
- Streamlit
- Pandas
- Plotly
